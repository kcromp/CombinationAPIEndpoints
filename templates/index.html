Combination API Endpoints

This web application contains 2 JSON API endpoints based in combination output.

/coins returns the number of ways pennies, nickels, dimes, and quarters can be combined to sum exactly $1.

/demo returns the number of combinations for an arbitrary set of currency names and denominations for an arbitrary total sum. The denomination_values.txt file accepts and parses the name of each denomination along with the number required of that denomination to reach the target sum.
